Privacy Policy for Tab Saver

Tab Saver is a browser extension that helps users save browser tabs locally, organize them into favorites, and reopen them later.

Last updated: March 10, 2026

Information the extension processes

Tab Saver may process the following information:

  • Tab titles
  • Tab URLs
  • Favorite and collection data created by the user
  • Language preference selected by the user

How the information is used

This information is used only to provide the core functionality of the extension, including:

  • Saving the current tab
  • Saving open tabs
  • Organizing saved tabs into favorites
  • Reopening saved favorites
  • Importing and exporting saved data
  • Remembering the user’s language setting

Data storage

Tab Saver stores user data locally in the browser using Chrome extension storage. The extension does not transmit user data to external servers.

Data sharing

Tab Saver does not sell, transfer, or share user data with third parties.

Remote code

Tab Saver does not use remote code. All code required to run the extension is packaged with the extension.

Permissions

Tab Saver uses the following permissions only for its stated functionality:

  • tabs: to read tab titles and URLs and reopen saved tabs
  • storage: to save favorites and user preferences locally
  • host permissions: to access tab URL and title information needed for saving tabs selected by the user

User control

Users can edit, delete, export, or import their saved data at any time within the extension.

Contact

If you have questions about this Privacy Policy, please contact through the support link on the Chrome Web Store listing or the project repository.

This page is intended for the Chrome Web Store listing of Tab Saver and describes the current data practices of the extension.
